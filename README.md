# tioc
tioc (***T**erminal **I**nput **O**utput **C**ontrol*)
a base for basic apps and libraries to use

# headers
* `tioc_color.h` for terminal colors
* `tioc_cursor.h` for terminal cursor manipulation
* `tioc_screen.h` for terminal screen manipulation
* `tioc_term.h` for general terminal manipulation

# what can i do with these?
* print text in colors (16, 256 and RGB supported)
* move, hide, show, save, and restore the cusor
* clear any line and screen in a give range
* bring the terminal to the alternate screen buffer
* turn off and on canonical input processing and input echoing
* get the size of the terminal
